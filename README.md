# 🛒 Mini E-commerce Website

This project is a **mini Amazon-style e-commerce website** built using **HTML, CSS, and JavaScript**.
It demonstrates how a basic online store works, including product display, search functionality, cart management, and a checkout interface.

---

## 🚀 Features

* 🧭 Amazon-style navigation bar
* 🔎 Product search functionality
* 🖼 Product images
* ⭐ Star ratings and reviews
* 🛒 Add to cart system
* ➕ Increase / ➖ decrease product quantity
* ❌ Remove items from cart
* 💰 Automatic total price calculation
* ✖ Close cart option
* 💳 Checkout page with address and payment form
* 💾 Cart saved using **localStorage**
* 📱 Responsive design for mobile devices
